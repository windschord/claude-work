# NFR: セキュリティ

## NS-001: 非rootユーザー

コンテナは非rootユーザー（`node` ユーザー、UID 1000）で実行されなければならない。

## NS-002: イメージサイズ

本番イメージはdevDependenciesを含まず、ビルド成果物のみを含まなければならない。
目標イメージサイズ: 500MB以下（node-pty等のネイティブバイナリ含む）。

## NS-003: 認証情報

GitHub ActionsワークフローはGITHUB_TOKENを使用してGHCRに認証し、
外部シークレット（Docker Hub認証情報等）を必要としないようにしなければならない。
