# US-003: デフォルト環境ラベルの廃止

## ユーザーストーリー

**As a** ClaudeWork管理者
**I want** 環境一覧から「デフォルト」ラベルが削除される
**So that** 不明瞭な「デフォルト」概念に惑わされずに環境を管理できる

## 受入基準

1. 環境カード（EnvironmentCard）に「デフォルト」バッジが表示されない
2. 環境セレクターのドロップダウンに「(default)」テキストが表示されない
3. すべての環境が削除可能（ただしセッション/プロジェクトから参照されている場合を除く）
4. 「デフォルト環境は削除できません」のメッセージが表示されない
