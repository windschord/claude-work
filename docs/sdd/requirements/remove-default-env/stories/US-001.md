# US-001: プロジェクト作成時の環境選択必須化

## ユーザーストーリー

**As a** ClaudeWorkユーザー
**I want** プロジェクト作成時に実行環境を明示的に選択する
**So that** 意図しない環境でセッションが実行されることを防げる

## 受入基準

1. プロジェクト追加モーダル（AddProjectModal）で環境を選択しなければ、「追加」ボタンが無効化されている
2. プロジェクト追加ウィザード（AddProjectWizard）で環境未選択の場合、次のステップに進めない
3. リモートリポジトリフォーム（RemoteRepoForm）で環境未選択の場合、送信ボタンが無効化されている
4. 環境が0件の場合、環境作成を促すエラーメッセージが表示される
5. デフォルト環境による自動選択が行われない
