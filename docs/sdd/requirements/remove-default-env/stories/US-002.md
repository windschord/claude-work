# US-002: セッション作成時の環境バリデーション

## ユーザーストーリー

**As a** ClaudeWorkユーザー
**I want** セッション作成時にプロジェクトの環境設定が必須となる
**So that** 環境未設定のまま誤ってセッションが作成されることを防げる

## 受入基準

1. プロジェクトに`environment_id`が設定されていない場合、セッション作成APIが400エラーを返す
2. エラーメッセージに「プロジェクトに実行環境が設定されていません」と表示される
3. デフォルト環境へのフォールバックが行われない
4. CreateSessionModalでプロジェクトの環境が未設定の場合、警告が表示される
