# US-002: CLAUDE.mdにプロセス実行テスト規則を追記

## ユーザーストーリー

開発者として、CLAUDE.mdにsubprocessテストのルールが明記されていることで、今後の開発で同様のバグを未然に防ぎたい。

## 要件

### 機能要件

- REQ-004: CLAUDE.mdに「プロセス実行のテスト規則」セクションを追加しなければならない
- REQ-005: `spawnSync`/`execSync`をモックする場合、`cwd`のアサーションを必須とし、`env`を明示的に渡している場合は`env`のアサーションも必須とするルールを記載しなければならない
- REQ-006: NG例とOK例をコード付きで記載しなければならない

### 記載内容

```markdown
### プロセス実行のテスト規則

- `spawnSync`/`execSync` をモックする場合、`cwd` のアサーションを必須とする

- `expect.objectContaining()` は省略したキーをスルーするため、重要なオプションは明示すること
- 子プロセスの `cwd` に `process.cwd()` を使う場合は、使用意図をコメントで明記すること
```
